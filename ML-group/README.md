- 书籍
  - 《机器学习》西瓜书 
    - 优点：网红，算法解析，适合学术，当字典用 
    - 缺点：没有实例，有大量习题（不会做。。。）
  - 机器学习实战
    - 优点：有实例！
    - 缺点：实例不够接地气
  - [《数据挖掘导论》](https://book.douban.com/subject/5286107/)
    - 优点：豆瓣高分，统计学经典入门书籍
    - 缺点：也都是理论
    - 点评亮点：所谓的数据挖掘不过是把N年来统计学的一些稀奇古怪的黑话转成图和表
